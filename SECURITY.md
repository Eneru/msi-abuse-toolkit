# Security Policy

## Purpose

This repository documents security research related to **Windows Installer (MSI/MSP) abuse techniques**.

Due to the sensitive nature of this research, this security policy exists to:
- protect contributors and users
- ensure responsible handling of information
- prevent accidental weaponization

---

## Supported Content

This project does **not** provide software releases intended for production use.

All content is provided **as-is**, for:
- research
- education
- defensive awareness

No guarantees are made regarding safety, completeness, or suitability.

---

## Reporting a Security Issue

If you identify a security concern related to this repository, including but not limited to:

- accidental weaponization
- unintended exploitability
- legal or compliance risks
- content violating GitHub ToS
- disclosure concerns

Please **do not open a public issue**.

Instead:
- Contact the repository owner directly via GitHub
- Provide a clear description of the concern
- Include references where applicable

---

## Zero-Day Research Handling

This repository may reference or discuss zero-day vulnerabilities under the following principles:

- No active exploitation guidance
- No ready-to-use payloads
- No targeting of unpatched systems
- Focus on understanding impact and mitigation

Any zero-day related content may be:
- redacted
- delayed
- removed

at the discretion of the maintainer.

---

## Responsible Use Disclaimer

The techniques described in this repository can be dangerous if misused.

By using this content, you agree that:
- you are responsible for your actions
- you will comply with applicable laws
- you will use this knowledge ethically

The maintainer is not responsible for misuse of this information.

---

## Changes to This Policy

This security policy may be updated at any time to reflect:
- changes in scope
- legal considerations
- platform requirements
- research maturity

Please review it periodically.
